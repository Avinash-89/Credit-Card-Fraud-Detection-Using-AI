# Credit-Card 💳-Fraud-Detection-Using-AI

## Overview
This project detects fraudulent financial transactions using machine learning and behavioral analysis.

## Features
- Fraud prediction using XGBoost
- Feature engineering (distance, time, amount)
- Interactive Streamlit dashboard
- Risk categorization (Low / Medium / High)

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit

## Key Concepts
- Imbalanced data handling
- Threshold tuning
- Feature engineering
- Real-world fraud patterns

## Run Locally
pip install -r requirements.txt
python train_model.py
streamlit run app.py
